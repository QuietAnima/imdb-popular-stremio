# IMDb Popular Stremio Addon

A self-hosted Stremio addon that serves IMDb popular movies and series as a Stremio catalog.

## Features

- Provides IMDb Most Popular Movies and TV Series catalogs to Stremio
- Auto-refreshes data daily to keep listings current
- Runs as a Docker container for easy deployment

## Setup

1. Clone this repository
2. Build and run with Docker:
   ```bash
   docker compose up -d
   ```
3. Install the addon in Stremio by navigating to the addon URL
